# Blogr Landing Page

A responsive landing page built as a Frontend Mentor challenge, completed as part of the Web Design Cohort Capstone Project.

## Links

- Live Site: 
- GitHub Repo: 

## Built With

- Semantic HTML5
- CSS Custom Properties (variables)
- CSS Grid & Flexbox
- Vanilla JavaScript (ES6+)
- Google Fonts (Overpass + Ubuntu)
- Mobile-first responsive design

## Features

- Fully responsive across mobile, tablet, and desktop
- Accessible navigation with ARIA attributes
- Mobile hamburger menu with toggle
- Dropdown menus in the navbar
- Keyboard navigation support (Escape key closes menus)
- Scroll-reveal animations using IntersectionObserver
- Semantic HTML structure
- Clean, maintainable CSS with custom properties

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser — no build tools needed
3. Or use VS Code with the **Live Server** extension for hot reload

## Deployment

Deploy for free using any of:
- [Netlify](https://netlify.com) — drag & drop the project folder
- [GitHub Pages](https://pages.github.com) — enable in repo Settings

## Design Decisions

- Used CSS custom properties for easy theming
- Implemented scroll-reveal with `IntersectionObserver` (no library needed)
- SVG icons kept inline/external for crisp rendering at all sizes
- CSS `clamp()` for fluid typography scaling
